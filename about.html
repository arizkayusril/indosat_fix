Indosat Ooredoo Hackdata 2019

Hackdata yang merupakan singkatan dari hackathon + datathon, adalah salah satu program Corporate Social Responsibility (CSR) Indosat Ooredoo di pilar Pendidikan sebagai upaya untuk mendorong anak muda Indonesia menciptakan solusi inovatif berbasis data analytics dengan menggunakan data set dan API dari Indosat Ooredoo.

Indosat Ooredoo menggandeng pemerintah provinsi DKI Jakarta dan Bandung, menyediakan total 346 data set (122 data set DKI Jakarta, 214 data set Bandung dan 10 data set Indosat Ooredoo) serta 19 API Indosat Ooredoo yang dapat digunakan dalam pengembangan solusi inovatif ini.

Topik yang dapat dipilih untuk solusi inovatif ini mencakup bidang: Pemerintahan Terbuka (Open Government), Kesehatan (Health), Kota Cerdas (Smart City) dan Bisnis (Business).

Diakhir program, 10 tim dengan solusi terbaik, akan diundang ke Jakarta untuk mempresentasikan langsung solusi inovatifnya di hadapan dewan juri untuk penilaian final.

Pendaftaran Hackdata dibuka pada tanggal 21 Oktober 2019 hingga 4 November 2019
